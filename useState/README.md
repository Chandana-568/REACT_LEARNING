# React `useState` Counter App

This is a simple React mini project created to understand the `useState` Hook.

The application includes a basic counter with three functionalities:

* ➕ Increment the count
* ➖ Decrement the count
* 🔄 Reset the count to zero

### What I Learned

* Using the `useState` Hook to manage component state.
* Updating state based on user interactions.
* Handling button click events in React.
* Re-rendering components when state changes.
* Building a simple interactive React application.

This project serves as my first hands-on practice with React state management and lays the foundation for building more interactive React applications.
