# 📝 React Todo App

A modern and interactive Todo App built with **React** to practice core React concepts such as state management, list rendering, conditional rendering, and CRUD operations.

## 🚀 Features

* ➕ Add new tasks
* ✏️ Edit existing tasks
* 💾 Save edited tasks
* ❌ Delete tasks
* ✅ Mark tasks as completed
* 🔍 Filter tasks (All, Active, Completed)
* 🧹 Clear all completed tasks
* 📱 Responsive and clean user interface

## 🛠️ Built With

* React
* JavaScript (ES6+)
* CSS3
* Vite

## 📚 React Concepts Practiced

* Functional Components
* JSX
* `useState` Hook
* Event Handling
* Conditional Rendering
* Rendering Lists with `map()`
* Updating Arrays in State
* Immutable State Updates
* Controlled Components
* Component Styling

## ▶️ Getting Started

### Clone the repository

```bash
git clone https://github.com/Chandana-568/REACT_LEARNING.git
```

### Navigate to the project

```bash
cd REACT_LEARNING
```

### Install dependencies

```bash
npm install
```

### Run the development server

```bash
npm run dev
```

Open the local URL shown in the terminal (usually `http://localhost:5173`) in your browser.
